# Camelot Projects 🛡️

Welcome to **Camelot Projects**, the digital Round Table. This repository serves as a centralized portal showcasing a curated suite of high-performance developer tools, libraries, and utilities designed for the modern web.

Every tool in the Camelot ecosystem is built on three core pillars:
1. **Performance**: Zero-bloat libraries with blazing fast execution speed.
2. **Security**: Hardened and verified compile/runtime configurations.
3. **Purity**: Clean design, standard APIs, and zero third-party dependencies.

## 🚀 Live Site
This repository is deployed as a GitHub Pages application.
Visit the live portal here: [https://justinthomsen.github.io/Camelot-Projects/](https://justinthomsen.github.io/Camelot-Projects/)

---

## 🗡️ The Arsenal
Our collection of software includes:
* **Excalibur**: An ultra-fast JavaScript bundler and compiler written in Rust.
* **Merlin**: A smart AI assistant providing IDE semantic search and predictions.
* **Lancelot**: A dependency and configuration SAST vulnerability scanner.
* **RoundTable**: A peer-to-peer WebRTC state synchronization framework.
* **Galahad**: A lightweight, strongly-typed state machine library.
* **HolyGrail**: A zero-config frontend packaging and automated deployment CLI.

## 🛠️ Local Development & Preview
To run this website locally, you can use any static server. For example:

### Using Python:
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000`.

### Using Node.js:
```bash
npx serve .
```
Then visit `http://localhost:3000`.
